# collection-tracker-ui
A collection tracker UI

## Requirements
- node v14 + npm v6
- pnpm v6

## Project setup
```bash
pnpm install
```

### Compiles and hot-reloads for development
```bash
pnpm run serve
```

### Compiles and minifies for production
```bash
pnpm run build
```

### Run your unit tests
```bash
pnpm run test:unit
```

### Run your end-to-end tests
```
pnpm run test:e2e
```

### Lints and fixes files
```
pnpm run lint
```
